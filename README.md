# Weather Diary (Дневник погоды)

## Автор
Халикова Элина Ильнуровна

## Описание
Приложение на Python для ведения дневника погоды. Позволяет добавлять, просматривать, фильтровать записи и сохранять их в JSON.

## Как запустить
1. Установите Python 3.x.
2. Скачайте файл `weather_diary.py`.
3. Запустите команду: `python weather_diary.py`

## Примеры использования
- Введите дату (YYYY-MM-DD), температуру и описание.
- Нажмите "Добавить запись".
- Используйте фильтры для поиска записей за конкретный день или с определенной температурой.
- Сохраните данные кнопкой "Сохранить в JSON".
